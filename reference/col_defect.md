# Colour text in an Defect-based theme.

Colour text in an Defect-based theme.

## Usage

``` r
col_defect(...)
```

## Arguments

- ...:

  Character strings, they will be pasted together with paste0(), before
  applying the style function.

## Value

A defect-themed colour function.

## Examples

``` r
col_defect("Defect-themed text")
#> <cli_ansi_string>
#> [1] Defect-themed text
```
