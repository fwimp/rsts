# Colour text in an Silent-based theme.

Colour text in an Silent-based theme.

## Usage

``` r
col_silent(...)
```

## Arguments

- ...:

  Character strings, they will be pasted together with paste0(), before
  applying the style function.

## Value

A silent-themed colour function.

## Examples

``` r
col_silent("Silent-themed text")
#> <cli_ansi_string>
#> [1] Silent-themed text
```
