# Colour text in an Necrobinder-based theme.

Colour text in an Necrobinder-based theme.

## Usage

``` r
col_necrobinder(...)
```

## Arguments

- ...:

  Character strings, they will be pasted together with paste0(), before
  applying the style function.

## Value

A necrobinder-themed colour function.

## Examples

``` r
col_necrobinder("Necrobinder-themed text")
#> <cli_ansi_string>
#> [1] Necrobinder-themed text
```
