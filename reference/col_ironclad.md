# Colour text in an Ironclad-based theme.

Colour text in an Ironclad-based theme.

## Usage

``` r
col_ironclad(...)
```

## Arguments

- ...:

  Character strings, they will be pasted together with paste0(), before
  applying the style function.

## Value

An ironclad-themed colour function.

## Examples

``` r
col_ironclad("Ironclad-themed text")
#> <cli_ansi_string>
#> [1] Ironclad-themed text
```
