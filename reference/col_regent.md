# Colour text in an Regent-based theme.

Colour text in an Regent-based theme.

## Usage

``` r
col_regent(...)
```

## Arguments

- ...:

  Character strings, they will be pasted together with paste0(), before
  applying the style function.

## Value

A regent-themed colour function.

## Examples

``` r
col_regent("Regent-themed text")
#> <cli_ansi_string>
#> [1] Regent-themed text
```
